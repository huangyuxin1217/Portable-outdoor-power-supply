# ESP32与STM32C8T6通信协议

## 📡 通信接口配置

### 硬件连接
| ESP32引脚 | 功能 | 连接到STM32 |
|----------|------|------------|
| GPIO16   | RX2  | STM32 TX   |
| GPIO17   | TX2  | STM32 RX   |
| GND      | 地   | STM32 GND  |

### 通信参数
- **串口**: UART2
- **波特率**: 115200 bps
- **数据位**: 8位
- **停止位**: 1位
- **校验位**: 无
- **流控**: 无

---

## 📦 数据帧格式

### 帧结构
```
+------+------+--------+------+------+-----+------+
| 帧头 | 长度 | 数据1  | ... | 数据N | 校验 | 帧尾 |
+------+------+--------+------+------+-----+------+
  2字节  1字节  N字节          1字节  2字节
```

### 字段说明
| 字段   | 长度   | 说明                           | 值/范围        |
|--------|--------|--------------------------------|----------------|
| 帧头   | 2字节  | 固定帧头标识                   | 0xAA 0x55      |
| 长度   | 1字节  | 数据域长度（不含帧头、校验、尾）| 0-255         |
| 数据域 | N字节  | 实际数据内容                   | 见数据定义     |
| 校验   | 1字节  | 校验和（长度+数据域累加和）    | 0x00-0xFF      |
| 帧尾   | 2字节  | 固定帧尾标识                   | 0x0D 0x0A      |

---

## 📊 数据定义

### 数据包内容（共12字节数据域）
| 序号 | 参数名称       | 数据类型 | 字节数 | 单位  | 数值范围    | 说明               |
|------|----------------|----------|--------|-------|-------------|--------------------|
| 1    | 输出电压反馈   | uint16_t | 2字节  | 0.01V | 0-6553.5V   | 高字节在前         |
| 2    | 电池电压采样   | uint16_t | 2字节  | 0.01V | 0-6553.5V   | 高字节在前         |
| 3    | 输出电流采样   | uint16_t | 2字节  | 0.01A | 0-655.35A   | 高字节在前         |
| 4    | 光敏传感器     | uint16_t | 2字节  | -     | 0-4095      | ADC原始值，高字节在前 |
| 5    | 温度           | int16_t  | 2字节  | 0.1℃  | -327.6-327.7℃| 高字节在前，有符号 |
| 6    | 交流市电采样   | uint16_t | 2字节  | 0.1V  | 0-6553.5V   | 高字节在前，RMS值  |

**总数据域长度**: 12字节

## 📝 完整帧示例

### 示例数据
- 输出电压: 12.34V → 1234 → 0x04D2
- 电池电压: 11.50V → 1150 → 0x047E
- 输出电流: 2.50A → 250 → 0x00FA
- 光敏传感器: 2048 → 0x0800
- 温度: 25.5℃ → 255 → 0x00FF
- 交流市电: 220.0V → 2200 → 0x0898

### 帧结构（十六进制）
```
AA 55                    // 帧头
0C                       // 长度 = 12字节
04 D2                    // 输出电压 = 1234 (12.34V)
04 7E                    // 电池电压 = 1150 (11.50V)
00 FA                    // 输出电流 = 250 (2.50A)
08 00                    // 光敏传感器 = 2048
00 FF                    // 温度 = 255 (25.5℃)
08 98                    // 交流市电 = 2200 (220.0V)
XX                       // 校验和
0D 0A                    // 帧尾
```

### 校验和计算
```c
校验和 = (0x0C + 0x04 + 0xD2 + 0x04 + 0x7E + 0x00 + 0xFA + 0x08 + 0x00 + 0x00 + 0xFF + 0x08 + 0x98) & 0xFF
```

## 🔄 通信流程

### STM32发送流程
1. 采集所有传感器数据
2. 按照数据定义组装数据包
3. 计算校验和
4. 添加帧头和帧尾
5. 通过UART发送完整数据帧
6. 建议发送频率: 100ms-1000ms

### ESP32接收流程
1. 监听UART2接收数据
2. 检测帧头 (0xAA 0x55)
3. 读取长度字段
4. 接收数据域
5. 读取校验和和帧尾
6. 验证校验和是否正确
7. 验证帧尾 (0x0D 0x0A)
8. 解析数据并显示/处理

## ⚠️ 错误处理

### 可能的错误类型
| 错误类型     | 处理方式                        |
|--------------|---------------------------------|
| 帧头错误     | 丢弃当前字节，继续搜索帧头      |
| 长度异常     | 丢弃整帧，重新等待              |
| 校验和错误   | 丢弃整帧，计数错误次数          |
| 帧尾错误     | 丢弃整帧，重新等待              |
| 超时         | 清空缓冲区，重新开始            |

### 超时设置
- **字节间超时**: 100ms（两个字节之间的最大间隔）
- **帧超时**: 500ms（从帧头到帧尾的最大时间）

## 📈 数据转换说明

### STM32端（发送）
```c
// 电压转换: 实际电压(V) × 100 → uint16_t
uint16_t voltage_data = (uint16_t)(actual_voltage * 100);

// 电流转换: 实际电流(A) × 100 → uint16_t
uint16_t current_data = (uint16_t)(actual_current * 100);

// 温度转换: 实际温度(℃) × 10 → int16_t
int16_t temp_data = (int16_t)(actual_temp * 10);

// 交流电压转换: 实际电压(V) × 10 → uint16_t
uint16_t ac_voltage_data = (uint16_t)(actual_ac_voltage * 10);
```

### ESP32端（接收）
```c
// 电压还原: uint16_t → 实际电压(V)
float voltage = data_u16 / 100.0f;

// 电流还原: uint16_t → 实际电流(A)
float current = data_u16 / 100.0f;

// 温度还原: int16_t → 实际温度(℃)
float temp = data_i16 / 10.0f;

// 交流电压还原: uint16_t → 实际电压(V)
float ac_voltage = data_u16 / 10.0f;
```

## 📌 注意事项

1. **字节序**: 所有多字节数据采用**大端模式**（高字节在前）
2. **数据对齐**: 无需特殊对齐，按顺序排列
3. **异常值**: 当传感器异常时，建议发送特定值（如0xFFFF）表示无效
4. **通信质量**: 建议添加通信成功率统计，便于调试
5. **调试模式**: 建议STM32和ESP32都提供串口输出用于调试验证
