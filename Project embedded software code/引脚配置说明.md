# ESP32 控制数学系统 - 引脚配置说明

## 项目信息
- **开发板**: MH-ET ESP32 DevKit
- **芯片**: ESP32-WROOM-32
- **开发环境**: PlatformIO + Arduino Framework

---

## 引脚配置表

### OLED显示屏 (I2C接口)
| 功能 | ESP32引脚 | 说明 |
|------|-----------|------|
| SDA | GPIO21 | I2C数据线 (默认I2C引脚) |
| SCL | GPIO22 | I2C时钟线 (默认I2C引脚) |
| VCC | 3.3V | 电源正极 |
| GND | GND | 电源地 |

**OLED规格**:
- 型号: SSD1306 / SSD1309
- 尺寸: 0.96寸 / 2.42寸
- 分辨率: 128x64
- I2C地址: 0x3C

### 串口通信

#### UART0 (USB串口 - 调试用)
| 功能 | ESP32引脚 | 说明 |
|------|-----------|------|
| TX | GPIO1 (TXD0) | USB转串口发送 |
| RX | GPIO3 (RXD0) | USB转串口接收 |

**用途**: 程序下载、串口调试监视器
**波特率**: 115200

#### UART2 (与 STM32 通信 - 预留)
| 功能 | ESP32引脚 | 说明 |
|------|-----------|------|
| TX | GPIO17 (TX2) | 发送数据给 STM32 |
| RX | GPIO16 (RX2) | 接收 STM32 数据 |

**用途**: 与 STM32 核心板进行数据交互
**状态**: 预留可用

## 注意事项

### 不可用或受限引脚
- **GPIO0**: 启动模式选择，启动时需为高电平
- **GPIO2**: 启动时需为低电平，板载LED
- **GPIO6-11**: 连接到Flash芯片，**禁止使用**
- **GPIO34-39**: 仅输入，无内部上拉/下拉电阻 

### 电源说明
- **工作电压**: 3.3V逻辑电平
- **供电方式**: USB 5V供电 (板载稳压至3.3V)
- **最大输出电流**: 单个GPIO约40mA，总电流不超过200mA

## 功能模块说明

### 1. OLED显示模块
**文件位置**: 
- `include/oled.h` - 头文件
- `src/oled.cpp` - 实现文件

**主要功能**:
- 显示文字、数字
- 绘制图形（线、矩形、圆形）
- 屏幕刷新控制

### 2. WiFi通信模块
**功能**: 
- STA模式连接路由器
- HTTP Web服务器
- 与手机APP进行数据交互

### 3. 串口通信模块（预留）
**用途**: 
- 外部传感器数据读取
- 与其他MCU通信
- 调试输出
